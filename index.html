Deploy Guide

GitHub → Vercel · Step by step

01
Install & Build Locally
Unzip the project, install dependencies, and confirm it builds without errors before pushing to GitHub.
# Unzip and enter directory cd dubai-riders npm install # Generate 3D models (requires Blender 3.6+) blender --background --python scripts/export_all_models.py # Create audio placeholders python scripts/create_audio_placeholders.py # Verify build npm run build # should complete with no errors
⚠ Node.js 18+ required
02
Create GitHub Repository
Go to github.com → New repository. Name it dubai-riders. Set it to Public (required for free Vercel). Do NOT initialise with README.
# In your project folder, initialise git git init git add . git commit -m "feat: initial Dubai Riders cinematic site" # Add your GitHub repo as remote (replace YOUR_USERNAME) git remote add origin https://github.com/YOUR_USERNAME/dubai-riders.git # Push to GitHub git branch -M main git push -u origin main
03
Connect to Vercel
Go to vercel.com → Sign in with GitHub → New Project → Import your dubai-riders repo. Vercel auto-detects Next.js.
# Vercel build settings (auto-detected, no changes needed) Framework Preset: Next.js Build Command: npm run build Output Directory: .next Install Command: npm install Node.js Version: 18.x
04
Add .gitignore (Critical)
Make sure these are excluded from your repo — especially /public/models/ if GLB files are large. Use Git LFS for binary assets over 100MB.
# .gitignore — add this to your project root node_modules/ .next/ .env.local .env*.local *.log # For large GLB files — use Git LFS git lfs install git lfs track "*.glb" git lfs track "*.hdr" git add .gitattributes
05
Auto-Deploy on Push
Every push to main triggers a Vercel rebuild automatically. Your site lives at dubai-riders.vercel.app — or connect a custom domain in Vercel settings.
# Future updates — just push and Vercel deploys automatically git add . git commit -m "update: new brand section" git push # → Vercel detects push → builds → live in ~60 seconds
✓ Your GitHub repo at arsalan807567 already has Vercel connected
06
Custom Domain (Optional)
In Vercel Dashboard → your project → Settings → Domains. Add your domain and point DNS to Vercel's nameservers. SSL is automatic.
# DNS settings at your registrar (e.g. Namecheap, GoDaddy) Type: A Name: @ Value: 76.76.19.61 # Vercel's IP Type: CNAME Name: www Value: cname.vercel-dns.com
