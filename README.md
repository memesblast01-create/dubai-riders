## рџЋЁ Technical Architecture

### Scroll System
- `useScrollController` hook reads `scrollTop` from a fixed overlay div
- Maps scroll position в†’ `currentSection` + `sectionProgress` (0вЂ“1)
- Drives `fromBrand`, `toBrand`, `transitionT` in Zustand store
- Zero GSAP ScrollTrigger dependency вЂ” pure scroll math for 60fps

### Bike Color Interpolation
```
paintMat.color.lerp(targetColor, smoothstep(transitionT))
```
Runs inside `useFrame` вЂ” GPU-synced, never drops frames

### Camera System
- Per-brand config: position + lookAt target
- Camera lerps to target with `delta * 1.6` speed
- Sinusoidal idle sway overlaid on top
- Parking lot uses wide angle (fov 58) overhead shot

### Post Processing Stack
1. **Bloom** вЂ” Headlights, emissives, street lamps glow
2. **Depth of Field** вЂ” Cinematic bokeh
3. **Film Grain** вЂ” Subtle noise texture
4. **Vignette** вЂ” Dark edges
5. **Chromatic Aberration** вЂ” Lens fringing
6. **ACES Filmic Tone Mapping** вЂ” Industry-standard HDRв†’LDR
